== Description ==

Fastest way to choose Featured Image to your Posts from Yahoo.com, Seznam.cz, etc..

= Showcase =

Even for the blind. The plugin also allows a blind user to enter a keyword to find a number of thumbnail suggestions and send this selection to the seeing assistant by e-mail. The assistant simply clicks on the best photo directly in the email and the photo is safely added to the article as Featured Image.
